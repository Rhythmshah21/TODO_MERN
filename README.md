## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Rhythmshah21/Todo_Mern.git
   
2. Move to client:

   ```bash
   cd Todo_Mern/client

3. Install dependencies:
   
   ```bash
   yarn

4. Start the development server:

   ```bash
   yarn start
   
5. Move to server:

   ```bash
   cd Todo_Mern/server

6. Install dependencies:
   
   ```bash
   yarn

7. Start the development server:

   ```bash
   yarn start
